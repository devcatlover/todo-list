#  Lista de Tarefas (To-Do List)

Este projeto é uma aplicação simples de lista de tarefas desenvolvida em **HTML**, **CSS** e **JavaScript**.  
O objetivo é praticar conceitos básicos de programação web, como manipulação do DOM e lógica de programação.

##  Funcionalidades
- Adicionar novas tarefas
- Remover tarefas ao clicar nelas
- Interface simples e responsiva

##  Estrutura do projeto
- `index.html` → Estrutura da página
- `style.css` → Estilos visuais
- `script.js` → Lógica de funcionamento
